## 배포 (GitHub Pages)

1. 새 repo 생성: `ai-study-1st`
2. 이 폴더 안의 내용물 전체를 repo 루트에 업로드
3. Settings → Pages → Source를 `main` 브랜치 / `/ (root)`로 설정
4. 몇 분 후 `https://[username].github.io/ai-study-1st/` 에서 열림
5. 이 URL을 사내 게시판에 공유

## 슬라이드 뷰어 조작

- 키보드: ← → (이동), Space (다음), Home (처음), End (마지막)
