구글 Gems를 통한 반복 업무 처리

💎 GEMS 개념
나만의 맞춤형
AI 인턴 만들기
제미나이에게 특정 페르소나와 지침을 부여해 반복 업무를 자동화하는 개인 맞춤형 AI 기틀.
"너는 이제부터 성평등 관점의 전문 기자/데스크야.
기사 초안을 주면 인권 침해 표현을 찾아 대안을 제시해."
🆓 반복 업무 자동화 🎯 출입처별 응용
젠더데스크 Gems 직접 사용해보기
📰 실제 활용: 부산일보 젠더데스크 Gems
❌ BEFORE
인권 침해·성평등 관점을 매번 수동 검토. 데스크 교열에 상당한 시간 소요.
✅ AFTER
기사 초안 입력 시 1초 만에 차별적 표현 탐지 및 대안 단어 자동 제안.
⏱ 교열 시간 단축
📈 보도 신뢰도 향상
♻️ 출입처별 응용

Gems 완성도 높이기 — 프롬포트 + 문서 업로드

✍️
정밀한 프롬포트
작성할수록 정확도 ↑
역할·페르소나 명확히 지정 — "너는 15년 경력 사회부 에디터야"처럼 구체적으로
원하는 출력 형식 명시 — "~다 체 스트레이트 기사, 400자 이내, 제목 포함"
예시(Few-shot) 포함 — 잘 쓴 기사 예시를 같이 넣으면 정확도 대폭 상승
제약 조건 명시 — "추측성 표현 금지", "익명 처리 필수" 등
📎
참고 문서 업로드
Gems의 '교과서' 역할
사내 보도 준칙·스타일 가이드 업로드 → 톤&매너 자동 준수
과거 우수 기사 샘플 삽입 → 해당 스타일 학습·재현
출입처별 전문 용어 사전 → 해수부·부산시 등 맥락에 맞는 정확한 단어 사용
문서가 많을수록 Gems 답변 품질이 기하급수적으로 향상
💡
프롬포트 작성이 막막하다면? — 제미나이에게 직접 부탁하세요
좋은 프롬포트를 쓰는 법을 몰라도 됩니다. 제미나이가 대신 써줍니다.
💬  "나는 부산일보 사회부 기자야. 내가 쓸 Gems 프롬포트를 작성해줘. 스트레이트 기사 초안을 빠르게 잡아주는 역할이야."

NotebookLM — 기자에게 최고의 AI 리서치 도구

✦ Google AI 리서치 도구
소스 기반 답변
할루시네이션 제로
업로드한 자료 안에서만 답변
AI가 마음대로 지어내지 않음
팩트만 · 추론 없음
🎯
업로드 소스 기반 답변 — 할루시네이션 없음
일반 AI와 달리 내가 올린 자료 안에서만 답변. 출처 인용까지 자동 표시 → 취재 팩트 검증에 최적.
📈
유료 구독 시 문서 처리량 5배 확대
무료 대비 최대 5배 많은 소스 동시 업로드·분석 가능. 대규모 취재 프로젝트·연속 기획 보도에 유리.
무료: 기본 용량 🚀 유료: 5× 확장
📁
멀티포맷 소스 업로드 — 취재 소스 전부 OK
정부 PDF, 유튜브 링크, 음성 녹음, 웹페이지 등 다양한 형식 한 곳에 모아 소스 기반 질문 가능.
📄 PDF ▶ YouTube 🎙 녹음 파일 🌐 웹페이지
✅
추론·창작 불가 → 기자에게 최고의 이유
AI가 "그럴 것 같다"고 지어내지 않음. 소스에 없으면 모른다고 답변 → 오보 위험 없는 신뢰형 리서치.
